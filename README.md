Um die Zellen auf der Bibliotheksseite, die Hörbücher anzeigen, aus VS Code heraus zu öffnen, müssen Sie Live Server installieren und die Seite über die Option „Mit Live Server öffnen“ im Kontextmenü von library.html öffnen.

Ich bin mir nicht sicher, warum im Header der Website gleichzeitig ein Hamburger-Menü und eine vollständige Menüleiste angezeigt werden. Ich habe beide so eingefügt, wie auf den Bildern zu sehen ist.

Eine Website wie diese sollte jedoch immer auf mindestens drei Bildschirmgrößen/Gerätetypen reagieren:

- Laptops/Desktop-Computer
- Tablets/iPads
- Smartphones

Vielleicht ist das Hamburger-Menü nur für die letzten beiden Gerätetypen vorgesehen (??). Jedenfalls macht das Hamburger-Menü gerade nichts mehr als Deko.

Aus Zeit Gründe habe ich keine weitere Media Queries im CSS eingefügt, außer die Einstellungen, die die Anzahl von Zellen pro Zeile regieren. Selbst das ist relativ primitiv.

Auch die Struktur der Daten in der .json-Datei ist ziemlich primitiv.

Das sind nur ein paar der vielen Punkte, die ich in der verfügbaren Zeit leider nicht angehen konnte!
